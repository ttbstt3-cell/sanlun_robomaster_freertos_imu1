# RoboMaster C板 SBUS 遥控车控制代码

基于 STM32F407IGHx 的 RoboMaster C板，使用 SBUS 协议接收遥控器信号，控制四个舵机（3个360度驱动轮 + 1个180度转向舵机）。

## 硬件配置

- **主控板**: RoboMaster C板 (STM32F407IGHx)
- **遥控器**: 乐迪遥控器 (SBUS信号)
- **舵机配置**:
  - 1号: 360度舵机 (大轮) - TIM1 CH1
  - 2号: 180度舵机 (转向) - TIM1 CH2
  - 3号: 360度舵机 (左小轮) - TIM1 CH3
  - 4号: 360度舵机 (右小轮) - TIM1 CH4

## 功能特点

### 1. 双模式控制

- **模式1 (驾驶模式)**: SwB 拨杆在下
  - 左摇杆上下 → 控制转向
  - 右摇杆上下 → 控制三个驱动轮前进/后退
  - 三个轮子按比例协同运动

- **模式2 (调试模式)**: SwB 拨杆在上
  - 右摇杆上下 → 单独控制1号大轮
  - 右摇杆左右 → 控制3号或4号小轮
  - SwE 拨杆 → 切换控制3号或4号轮

### 2. 物理零点补偿 (TRIM)

解决360度舵机硬件公差问题：
- 现象：摇杆居中时轮子缓慢转动
- 解决：使用 TRIM_M1/M3/M4 补偿值
- 调整方法：
  - 轮子后退 → 增大TRIM值
  - 轮子前进 → 减小TRIM值

### 3. LED 状态指示

- 蓝灯 (PH10): 系统心跳，闪烁
- 绿灯 (PH12): SBUS信号正常时常亮
- 红灯 (PH11): 信号丢失时亮起

### 4. 失控保护

信号丢失时自动执行：
- 发送TRIM补偿值保持轮子静止
- 转向舵机锁定在中位

## 通道映射

| CH | 遥控器输入 | 功能 |
|----|------------|------|
| CH1 | 右摇杆左右 | 调试模式控制小轮 |
| CH2 | 右摇杆上下 | 主驱动/油门 |
| CH3 | 左摇杆上下 | 转向控制 |
| CH5 | SwE | 选择3号/4号轮 |
| CH9 | SwB | 驾驶/调试模式切换 |

## SBUS 配置

- 波特率: 100000
- 数据位: 9位
- 停止位: 2位
- 校验: 偶校验 (EVEN)

## 使用说明

1. 使用 Keil MDK 打开 `sanlun_robomaster.uvprojx` 工程
2. 编译并下载到 RoboMaster C板
3. 打开遥控器，确认 SBUS 信号正常
4. 根据需要调整 TRIM 值使轮子静止

## 调参步骤

1. 确保遥控器开机，SBUS信号正常（绿灯亮）
2. 不要触碰任何摇杆
3. 观察哪个轮子在转动
4. 修改 main.c 中的 TRIM 值：
   - `TRIM_M1` → 1号轮
   - `TRIM_M3` → 3号轮
   - `TRIM_M4` → 4号轮
5. 重新烧录，直到三个轮子都静止
